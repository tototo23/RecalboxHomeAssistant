This repository allows you to integrate Recalbox in your Home Assistant :

  • In your dashboard : Get status, Display current game, Turn off, Reboot, Take screenshots, ...
  • Trigger any automation you want. For example : change light color according to the game launched, send notifications, etc.
  • Voice/text commands with assistant : Launch a game, Ask what is the current game, Stop current game, Take a screenshot, Turn off Recalbox, etc
